TitanDriver
  titandrv.pas - основной файл исходного кода драйвера
  datastorage.pas - дополнительный файл, библиотека хранилища данных
  JsonStorage.pas - дополнительный файл, формат JSON для хранилища
  TitanDriverTest.lpr - файл проекта программы тестирования драйвера
также требуется внешняя библиотека Ararat Synapse:
  http://www.ararat.cz/synapse/doku.php/download

titan.drv - папка проекта драйвера ФР Титан для R-Keerer
Testing.drv - папка проекта стандартного тестового драйвера для R-Keerer
